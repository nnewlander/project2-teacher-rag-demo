适用对象：教研支持。适用课程阶段：Python进阶。所属模块：教师工作台。文档版本：v1.1。
本资料用于支持教师在课堂演示场景下快速定位问题，重点覆盖 练习讲解、坐标系 等高频知识点。
常见教学问题：1）学生听懂概念但写代码时容易漏掉细节；2）课堂演示时老师需要快速解释报错；3）备课时希望找到可直接复用的示例。
处理建议：先确认问题属于语法理解、平台使用还是课程设计，再判断是否需要展示最小可运行代码。若涉及课堂演示，建议优先给出步骤化解释。
示例代码片段：
print('hello')

扩展示例：
import turtle
turtle.goto(50, 100)
turtle.dot(10)
常见误区：把概念解释成结果、忽略运行环境、只给答案不给原因。推荐教师先定位触发条件，再给出修正方法和课堂讲法。
检索标签：坐标系、练习讲解、函数定义。关联场景：课中答疑、作业讲评、课后技术支持。
