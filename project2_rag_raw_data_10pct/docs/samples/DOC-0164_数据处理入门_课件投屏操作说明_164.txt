常见教学问题：1）学生听懂概念但写伐码时容易漏掉细节；2）课堂演示时敎师需要快速解释报错；3）备课时希望找到可直接复用的示例。
处理建议：先确认问题属于语法理解、平合使用还是课程设计，再判断是否需要展示最小可运行代码。若涉及课堂演示，建议优先给出步骤化解释。
示例代码片段：
print('hello')

扩展示例：
def add(a, b):
    return a + b
print(add(2, 3))
常见误区：把概念解释成结果、忽略运行环境、只给答案不给原因。推荐教师先定位触发条件，再给出修正方法和课堂讲法。
